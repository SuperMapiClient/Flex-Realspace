类public class FlyingOperator
继承FlyingOperator ClassBase flash.events.EventDispatcher

飞行状态类 （不可以初始化，对外开放） 该类作为 Scene 类的属性，主要用于设置场景的飞行状态，包括指定飞向下一个场景的方位、控制飞行方式以及飞行动画。 该类不能创建，只能通过场景获取 Scene.get_flyingOperator()

示例
以下例子示范了如何使用 
         //获取Realspace控件的场景，控件和场景是一对一的捆绑关系
         scene=m_sceneControl.get_scene();
         //获取飞行状态对象
         var flyingOperator:FlyingOperator=scene.get_flyingOperator();
         //新建一个照相机对象
         var camera:SuperMap.Web.Realspace.Camera=new SuperMap.Web.Realspace.Camera(116.348,40,1000000.0);
         //按照范围飞行定位
         flyingOperator.flyToBounds(new SuperMap.Web.Core.Rectangle2D(110,35,111,36),5000);

flyTo()方法
public function flyTo(camera:Camera, nMillSec:Number = -1, flyingMode:Number = 6):void

在指定的时间内，以指定的方式从当前场景飞行至指定的相机位置。

参数

camera:Camera — 相机类，设置场景将要飞至的位置。
 
nMillSec:Number (default = -1) — 飞行时间，单位为毫秒。可选参数，默认值为-1，将以一个默认速度匀速飞行到指定地点。
 
flyingMode:Number (default = 6) — 飞行方式。默认值为 SuperMap.Web.Realspace.FlyingMode.CAMERA_NORMAL。（也就是6）

flyToBounds()方法 
public function flyToBounds(rectangle2D:Rectangle2D, nMillSec:Number = -1):void

将当前场景飞行至指定的 Bounds 区域

参数

rectangle2D:Rectangle2D — 将指定的 bounds 区域作为显示区域在场景中显示。该类为 Rectangle2D 类型，单位为经纬度。
 
nMillSec:Number (default = -1) — 飞行时间，单位为毫秒。可选参数，默认值为-1，将以一个默认速度匀速飞行到指定地点。

flyToGeometry()方法 
public function flyToGeometry(geoObject:Geometry3D, nMillSec:Number, flyingMode:Number):void

在指定的时间内，以指定的方式从当前场景飞行至指定的几何对象。

参数

geoObject:Geometry3D — 三维几何对象。
 
nMillSec:Number — 飞行时间，单位为毫秒。可选参数，默认值为-1，将以一个默认速度匀速飞行到指定地点。
 
flyingMode:Number — 飞行方式。默认值为SuperMap.Web.Realspace.FlyingMode.MULTIPOINTFLY_NORMAL。
