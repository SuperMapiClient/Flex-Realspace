包SuperMap.Web.Realspace
类public class Scene
继承Scene ClassBase flash.events.EventDispatcher

三维场景类 （不可以初始化，对外开放） 该类主要用于装载影像、模型、地形图层、屏幕图层、跟踪图层、飞行对象等。当加载三维场景时存在一个默认的三维场景。 有关 Scene 的详细信息请参见专题文档 三维场景控件和三维场景。 该类不可创建，只能通过 SceneControl.get_scene() 获取。

示例
以下例子示范了如何使用 
         //获取Realspace控件的场景，控件和场景是一对一的捆绑关系
         scene=m_sceneControl.get_scene();
         //新建一个照相机对象，把照相机的位置设置为中国的中心点
         var camera:SuperMap.Web.Realspace.Camera=new SuperMap.Web.Realspace.Camera(104.348,28.708,10288740.0);
         //将场景的视点移动到中国区域
         scene.get_flyingOperator().flyTo(camera,5000);

findSelection3Ds()方法 
public function findSelection3Ds(hasObjectSelected:Boolean = false):Array

查找当前三维场景中图层的选择集。该方法返回的是 Selection3D 类对象的数组， 每一个 Selection3D 类对象对应三维场景中一个图层中所有处于选中状态的对象集合，若没有对象处于选中状态，则数组为空。

参数

hasObjectSelected:Boolean (default = false) — 指定返回的选择集是所有图层的选择集，还是选择集对象不为空的图层选择集。当其为 true 时，返回选择集对象不为空的图层选择集，否则返回所有图层的选择集。

返回
Array — 返回 Selection3D 类对象的数组

open()方法 
public function open(sceneAddressUrl:String, sceneName:String):Boolean

打开指定场景及场景中的所有图层。 加载指定场景时，同时加载场景的环境变量包括环境光，背景色，地形夸张系数，相机，雾效，星空，大气层，经纬网等以及场景中的图层数据。

参数

sceneName:String — 场景名称。

返回
Boolean — 返回是否成功加载场景，true 表示成功加载，否则返回 false。
