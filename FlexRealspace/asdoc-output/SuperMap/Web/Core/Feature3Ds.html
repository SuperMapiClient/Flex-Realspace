三维地理要素类集合 三维要素集合类不单纯是三维要素对象的集合，它是树状结构，也就是说三维要素集合对象还可以包含子对象， 子对象也是三维要素集合类的对象。因此，一个三维要素集合对象可以包括零个或多个子对象和零个或多个三维要素对象。

findFeature3D()方法
public function findFeature3D(index:Object, option:Number):Feature3D

根据指定查找方式查找指定要素对象。

参数

index:Object — 按要素对象的 ID 号（Number类型）或名字（String类型）查找要素对象。
 
option:Number — 要素对象的查找方式常量，分为 AllFeatures 和 TopFeaturesOnly。Feature3DSearchOption 枚举, AllFeatures 遍历主干和分支中所有对象进行查找；TopFeaturesOnly 表示只遍历主干中的对象进行查找。
