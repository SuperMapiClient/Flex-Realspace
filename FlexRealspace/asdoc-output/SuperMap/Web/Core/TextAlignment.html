类public class TextAlignment
继承TextAlignment Enum Object

文本对齐方式常量 （模拟枚举，不可以初始化，对外开放） 文本对齐常量指定文本中各子对象的对齐方式。文本对象的每个子对象的位置是由文本的锚点和文本的对齐方式共同确定的。 当文本子对象的锚点固定，对齐方式确定文本子对象与锚点的相对位置，从而确定文本子对象的位置。
