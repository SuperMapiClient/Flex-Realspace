Rectangle2D()构造函数
public function Rectangle2D(left:Object = 0, bottom:Number = 0, right:Number = 0, top:Number = 0)

Rectangle2D 构造函数。在地理坐标系下，通过左边界、右边界经度和上边界、下边界纬度来构造矩形对象。 在构造矩形对象时，左边界值应小于右边界值，下边界值应小于上边界值。

参数
left:Object (default = 0) — 地理坐标系下矩形左边界经度值。
 
bottom:Number (default = 0) — 地理坐标系下矩形下边界的纬度值。
 
right:Number (default = 0) — 地理坐标系下矩形右边界的经度值。
 
top:Number (default = 0) — 地理坐标系下矩形上边界的纬度值。

示例
以下例子示范了如何使用 
         //创建一个矩形对象
         var rectangle2D:SuperMap.Web.Core.Rectangle2D=new SuperMap.Web.Core.Rectangle2D(116,38,120,40);
         //判断句型中是否包含点（119,39），返回true
         var isContains:Boolean= rectangle2D2.contains(119,39);
