CallBackEvent()构造函数
public function CallBackEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, data:Object = null)

回调事件信息类构造函数

参数
type:String — 事件的类型。
 
bubbles:Boolean (default = false) — 确定 CallBackEvent 对象是否参与事件流的冒泡阶段。默认值为 false。
 
cancelable:Boolean (default = false) — 确定是否可以取消 CallBackEvent 对象。默认值为 false。
 
data:Object (default = null) — CallBackEvent 对象存储的信息。
